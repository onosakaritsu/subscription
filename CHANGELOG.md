# 更新日志

## 1.0.0 - 2026-07-06

- 正式可用版：完成本地优先订阅管理、统计、提醒、备份恢复和数据保护闭环。
- 完整订阅管理功能：新增、编辑、删除、快速启用/停用、确认已续费、复制订阅。
- 完成统一状态色标、币种显示优化、续费日历和 ICS 日历导出。
- 完成手动备份、备份下载、内部备份恢复、外部备份恢复和恢复前保护。
- 新增普通 JSON 导入预览、外部备份恢复预览和差异摘要。
- 增强数据完整性检查，明确错误、警告和只读检查边界。
- 补齐 favicon、版本显示、`/api/health` 版本返回和启动脚本最终优化。
- 补充隐私说明 `PRIVACY.md`、发布检查清单 `RELEASE_CHECKLIST.md`。
- 保持 Apache License 2.0、免责声明和中文优先开源文档。
- 完成测试、API 验证和真实浏览器截图验收。

## 2.0.0 - 2026-07-02

- 从早期 macOS SwiftUI / WidgetKit 方向迁移为本地优先 Web App。
- 新增零第三方依赖的 Node.js 原生 HTTP 后端。
- 新增静态 HTML/CSS/JavaScript 前端。
- 新增本地 JSON 文件存储、导入和导出。
- 新增订阅日期计算、筛选和摘要统计。
- 移除 WidgetKit 与 App Group 依赖。
- 新增开源项目文件。

## 2.1.0 - 2026-07-05

- 新增续费状态标签：已过期、今日续费、3 日内、7 日内、本月内、正常、已停用。
- 优化订阅列表默认排序，启用订阅优先并按续费日期从近到远展示。
- 新增首页统计卡片，支持多币种月度和年度折算。
- 增强最近续费摘要，优先展示紧急续费项目。
- 新增本地自动备份，最多保留最近 20 份。
- 优化 JSON 导出文件名。
- 新增一键启动脚本 `start-subscription-manager.sh`。
- 优化 420px～600px 窄窗口下的桌面化使用体验。
- 将 GitHub 默认文档入口调整为中文，并新增独立英文 README。

## 2.2.0 - 2026-07-05

- 新增备份列表、备份预览和备份恢复 API。
- 新增前端“备份与恢复”区域，支持确认后恢复备份。
- 恢复备份前自动创建恢复前备份。
- 导入 JSON 前自动保护当前数据，并增强失败提示。
- 新增轻量续费日历视图，展示本月和下月续费项目。
- 优化一键启动脚本，增加 Node.js 版本检查。
- 补充 Phase 3 测试覆盖。
